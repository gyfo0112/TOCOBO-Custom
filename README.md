# TOCOBO 클론 코딩 🧴

실제 화장품 브랜드 TOCOBO 공식 웹사이트를 React로 클론 코딩한 프로젝트입니다.

## 📌 프로젝트 소개
브랜드 아이덴티티에 맞는 UI를 구현하고
Hero / Collection / SkinCare / Instagram 등
다양한 섹션과 인터랙션을 직접 구성했습니다.

## ⚙️ 주요 기능
- TopBanner: 자동 재생 슬라이더 + 닫기 기능
- Header: 스크롤 감지 스타일 변경 + 모바일 메뉴
- Nav: 섹션 부드러운 스크롤 이동 (커스텀 훅)
- Hero: 풀스크린 이미지 슬라이더 + 커스텀 네비게이션
- Collection: 카테고리 카드 슬라이더
- SkinCare: 제품 이미지 슬라이더
- Instagram: 피드 이미지 그리드
- AOS 스크롤 애니메이션
- 최상단 이동 고정 버튼

## 🛠 사용 기술
- React (useState, useEffect, useRef, useCallback)
- Swiper
- AOS (Animate On Scroll)
- SCSS
- Vite

## 🚀 실행 방법
```bash
npm install
npm run dev
```
